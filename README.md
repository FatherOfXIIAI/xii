## 🚀 Quick Start

### Prerequisites

-   Python 3.9+
-   Node.js 18+
-   pnpm

> **Note for Windows Users:** Consider WSL 2 for smoother compatibility.

### Start with XII AI Starter

```bash
git clone https://github.com/xii-ai/xii-starter.git

cp .env.example .env

pnpm i && pnpm build && pnpm start
```

### Manually Build XII AI

```bash
git clone https://github.com/xii-ai/xii.git

# Checkout the latest stable release
git checkout $(git describe --tags --abbrev=0)
```

### Edit the Configuration

Copy the `.env.example` to `.env` and add your specific values.

```
cp .env.example .env
```
